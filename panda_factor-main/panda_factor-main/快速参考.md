## 📋 常见问题

### 端口被占用
```bash
py port_manager.py
# 选择: 1 (自动清理)
```

### Web界面空白
访问新界面：
```
http://127.0.0.1:8111/factor/factor_library.html
```

### AI不响应
1. 检查服务是否运行
2. 查看终端错误信息
3. 测试API: http://127.0.0.1:8111/docs

## 💡 快速示例

### 生成因子代码
1. 打开因子库界面
2. 切换到"代码生成"标签
3. 输入描述，点击"生成代码"

### AI对话
1. 切换到"AI助手"标签
2. 选择合适的模型
3. 输入问题，点击"发送"

## 📁 重要文件

| 文件 | 说明 |
|------|------|
| `启动因子库.bat` | 一键启动 |
| `start_complete.py` | 完整启动脚本 |

| `config.yaml` | 配置文件（API密钥） |

## 🎯 配置文件位置

```
panda_common/panda_common/config.yaml
```

包含：
- LLM API密钥（3个）
- 模型配置（4个模型）
- MongoDB配置
- 负载均衡策略

## ⚠️ 注意事项

1. **保持终端运行** - 关闭终端会停止服务
2. **MongoDB可选** - 没有也能运行
3. **API密钥** - 已配置3个密钥轮询
4. **浏览器** - 推荐Chrome或Edge
