# SCGU Lightweight Edition - 个人轻量级版本

## 🎯 概述

轻量级版本专为个人开发者、学术研究者和小型团队设计，提供核心功能，易于上手。

## 📦 核心特性

- ✅ 自监督预训练 (单机版)
- ✅ 基础多模态支持
- ✅ SCGU遗忘算法
- ✅ 简化的API
- ✅ 完整文档和示例
- ✅ 开源免费

## 💻 系统要求

- Python 3.8+
- GPU: NVIDIA GTX 1060+ (可选)
- RAM: 16GB+
- Storage: 100GB+

## 🚀 快速开始

```bash
# 安装
pip install -r requirements-lightweight.txt

# 运行示例
python examples/quick_start.py

# 训练模型
python train.py --mode ssl --epochs 50
```
